Karatsuba algorithm

Given two long integers \( 1 \le x,y \le 10^{50000} \) in decimal, output their product \( xy \).

Please note that some programming languages (like Haskell, Python, Java) provide long arithmetic out of the box. Please do not use built-in functions, but implement your own procedure.


Sample Input:
239030239030566179
56617956617923930
Sample Output:
13533403703804583268316967452763470


Memory limit: 256 Mb
Time limit: 5s
