Create and print (using the print function) an array of class np.ndarray with width 4 and height 3 with 2s on the main diagonal and 1s on the first diagonal above the main one, i.e. realization of the matrix

\[ \begin{pmatrix}2 & 1 & 0 & 0 \\ 0 & 2 & 1 & 0 \\ 0 & 0 & 2 & 1 \end{pmatrix} \]

Please note that we use Python 3, where print() accepts arguments in brackets.

Hint: the use of addition and multiplication by constant can be an easy way.



Memory limit: 256 Mb
Time limit: 5s
