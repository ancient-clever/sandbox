Given the natural number \( 1 \le n \le 10^5 \) and the array of integers \( A[1 \ldots n] \), not exceeding \( 10^9 \) by absolute value. Output \( 1 \), if the array \( A \) contains the number occurring strictly more than \( n/2 \) times, and output \( 0 \) otherwise.


Sample Input:
5
2 3 9 2 2
Sample Output:
1


Memory limit: 256 Mb
Time limit: 5s
