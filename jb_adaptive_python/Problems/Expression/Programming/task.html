Write a function to compute the value of the following expression:

\[ \dfrac{a + bc}5 - a^3 + \dfrac{a + \frac c3} 4 \]

Where a, b, c - integers.
The function should have a name calc_expr and the three parameters: a, b, c.


Sample Input:
1 2 -3
Sample Output:
-2.0


Memory limit: 256 Mb
Time limit: 5s
