Sets of symbols

In this problem, we will learn to specify sets of symbols.

To do this we use the wildcards [ ].

Inside the brackets we can list all the characters that suit us: we can write a regular expression "[abc]bc", which will fit the strings abc, bbc, and cbc:

import re
results = re.findall("[abc]bc", "abc bbc cbc abb")
print(results)


Entry [abc] should be considered as a single symbol, which describes: either a, or b, or с.

The other way to define a set of symbols is the alteration wildcard |.

Let’s describe a group of symbols inside the parentheses, and use the alteration symbol: "(a|b|c)bc"

import re
results = re.findall("(a|b|c)bc", "abc bbc cbc abb")
print(results)


Unlike a set of symbols specified by the wildcards [ ], the alteration sign can be used for non-single-symbol sequences:  the pattern "(abc|abe|dd)x" this way will fit the three lines – "abcx", "abex" and "ddx".
