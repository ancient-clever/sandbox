Implement a function to compute the double factorial, that is the product of natural numbers with the same parity and not exceeding a given number. 

For example:
7!!=7⋅5⋅3⋅1
8!!=8⋅6⋅4⋅2

The function argument can be any non-negative integer.


Sample Input:
7

Sample Output:
105


Memory limit: 256 Mb
Time limit: 5s
