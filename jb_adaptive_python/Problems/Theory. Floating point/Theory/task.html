Floating-point. A type that represents numbers with fractional parts.

If you are not sure what type a value has, the interpreter can tell you. 
>>> type(17.6)
<type 'float'>
Not surprisingly,  it belongs ﻿to the type float.
﻿
Float type support the following operations, sorted by ascending priority:
        x + y            sum of x and y
        x - y            difference of x and y
        x * y            product of x and y
        x / y            quotient of x and y
        x // y           (floored) quotient of x and y
        x % y           remainder of x / y
        - x                  x negated
        + x                 x unchanged
        abs( x )         absolute value or magnitude of x
        int( x )           x converted to integer
        long( x )         x converted to long integer
        float( x )       x converted to floating point
        pow( x , y)    x to the power y
        x ** y           x to the power y
﻿

Conversion from floating point to integer may round or truncate as in C; see functions math.floor() and math.ceil() for well-defined conversions.﻿
