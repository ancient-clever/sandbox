You are given two coordinates on a chess board of the first queen and two of the second. Find out, whether two queens hit each other or not.

INPUT
Four integer numbers \( x_1, y_1, x_2, y_2 \) are being typed.

OUTPUT
Type "YES" (uppercase) if they hit each other or "NO" if the don't.

Sample Input:
1 1 3 3
Sample Output:
YES

Sample Input:
1 1 4 3
Sample Output:
NO

Sample Input:
2 2 5 2
Sample Output:
YES


Memory limit: 256 Mb
Time limit: 5s
