Your program will have an access to foo function, which can throw exceptions.

You need to write a code, which launches this function and than catches the ArithmeticError, AssertionError, ZeroDivisionError exceptions and outputs the name of the caught exception.

An example of the solution, which you need to send for checking.

try:
    foo()
except Exception:
    print("Exception")
except BaseException:
    print("BaseException")

Hint: see https://docs.python.org/3/library/exceptions.html#exception-hierarchy
