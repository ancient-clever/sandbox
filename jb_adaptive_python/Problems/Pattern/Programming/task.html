The simplest example of a pattern is a string that contains the characters that we want to find. For example, the string "abc" is the pattern for the string "abc", thus

import re
results = re.findall("abc", "abcAbcabc")
print(results)

Outputs all occurrence of the string "abc" in the string "abcAbcabc"
