Pairwise different summands

Given an integer \( 1 \le n \le 10^9 \) find the maximal number \( k \) such that \( n \) can be represented as a sum of pairwise different positive integers. In the first line output \( k \), in the next line output \( k \) summands.


Sample Input:
4

Sample Output:
2
1 3

Sample Input:
6

Sample Output:
3
1 2 3


Memory limit: 256 Mb
Time limit: 5s
