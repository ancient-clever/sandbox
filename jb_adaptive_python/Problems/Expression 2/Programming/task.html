Write a function to compute the value of the following expression:

\[ \dfrac{1 - 6pq}{pq} \]

The function should have the name calc_expr and a single parameter \( p \in (0, 1) \). It may be convenient to create the variable \( q = 1 - p \) inside the function.


Sample Input:
0.5
Sample Output:
-2.0


Memory limit: 256 Mb
Time limit: 5s
