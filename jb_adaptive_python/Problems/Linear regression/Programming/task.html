Find the optimal coefficients to construct a linear regression.

On input your program will get the name of the csv-file, from which you need to read data. You can view an example here. They can be downloaded as follows:

fname = input()  # read file name from stdin
f = urllib.request.urlopen(fname)  # open file from URL
data = np.loadtxt(f, delimiter=',', skiprows=1)  # load data to work with

Your task is to calculate the vector of the coefficients of the linear regression for the prediction of the first variable (the first column of data) by all the rest. Please remember that linear regression model is \( y = \beta_0 + \beta_1 x_1 + \cdots + \beta_n x_n \).

Print the coefficients of the linear regression, starting with \( \beta_0 \), space separated. We will verify the match with an accuracy of up to 4 digits after the decimal point.

The methods and techniques that can help you in the process of solution (may be hints!):
