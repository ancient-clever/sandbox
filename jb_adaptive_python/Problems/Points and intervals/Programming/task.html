Points and intervals

The first line contains the two integers: \( 1 \le n \le 50000 \) and \( 1 \le m \le 50000 \) — the amount of intervals and points on the straight line, respectively. Next \( n \) lines each contain the two integers \( a_i \) and \( b_i \) (\( a_i \le b_i \)) — the coordinates of interval ends. The last line contains \( m \) integers — points positions. All coordinates do not exceed \( 10^8 \) by absolute value. The point is considered belonging to the specified interval, if it lies inside this interval or on the interval boundary. For each of the points output to how many intervals it belongs to, in the order of occurence of these points in the input.


Sample Input:
2 3
0 5
7 10
1 6 11
Sample Output:
1 0 0


Memory limit: 256 Mb
Time limit: 1s
