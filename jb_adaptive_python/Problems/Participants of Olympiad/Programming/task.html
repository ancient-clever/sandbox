During the Olympiad, each of the participants received his own identification number – an integer. The list of participants of the Olympiad needs to be sorted by the number of points they have gained.

Input data

The first line contains the integer N (1 ≤ N ≤ 1000) – the number of participants. On each of the following line is given an identification number and the number of points gained by the corresponding participant. All numbers in the input file do not exceed 100000.

Output data

Output the same list of participant as in input, but in descending order of points gained. If some of the participants have the same number points, they should be put in ascending order of their identification numbers.


Sample Input:
3
101 80
305 90
200 14
Sample Output:
305 90
101 80
200 14

Sample Input:
3
20 80
30 90
25 90
Sample Output:
25 90
30 90
20 80


Memory limit: 256 Mb
Time limit: 1s
