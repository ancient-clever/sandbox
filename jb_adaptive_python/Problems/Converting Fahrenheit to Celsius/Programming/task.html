Converting Fahrenheit to Celsius

Write a Python program to convert Fahrenheit to Celsius.

\( C = 5/9 (F - 32) \) 


Sample Input:
84

Sample Output:
84.0 degrees Fahrenheit is 28.88888888888889 degrees Celsius

Sample Input:
128

Sample Output:
128.0 degrees Fahrenheit is 53.333333333333336 degrees Celsius

Sample Input:
-122

Sample Output:
-122.0 degrees Fahrenheit is -85.55555555555556 degrees Celsius


Memory limit: 256 Mb
Time limit: 5s
