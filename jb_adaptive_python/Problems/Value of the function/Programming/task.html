Write the function f(x), which returns the value of the next function, defined for any number:

\[ f(x)= \begin{cases}   1 - (x + 2)^2,\quad &\text{at } x\le -2\\  -\frac x2 ,\quad &\text{at } -2 \lt x \le 2\\   (x-2)^2 + 1,\quad &\text{at } 2 \lt x \end{cases} \]

You need only to implement the function; the solution should not make any input-output operations.


Sample Input:
4.5
Sample Output:
7.25

Sample Input:
-4.5
Sample Output:
-5.25

Sample Input:
1
Sample Output:
-0.5


Memory limit: 256 Mb
Time limit: 5s
