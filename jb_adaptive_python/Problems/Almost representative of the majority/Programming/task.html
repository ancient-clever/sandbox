Given the natural integer \( 1 \leq n \leq 10^5 \) and the sequence of integers \( 0 \leq a_1, \dotsc, a_n \leq 10^9 \). Output 1, is the sequence contains an integer, occurring strictly more than \( \frac{n}{3} \) times, and 0 otherwise.


Sample Input:
5
2 3 9 1 2
Sample Output:
1


Memory limit: 256 Mb
Time limit: 5s
