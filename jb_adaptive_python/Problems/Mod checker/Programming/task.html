Lambda functions provide us with a convenient way to create a function "in place". Nevertheless, sometimes when you need to create many similar lambda functions, it will be even easier to create a function that will generate them.

Implement the mod_checker(x, mod=0) function, which will generate lambda function of the single argument y, which will return True, if the remainder of dividing y by x equals to mod, and False otherwise.

Example:


  
mod_3 = mod_checker(3)
print(mod_3(3)) # True
print(mod_3(4)) # False
mod_3_1 = mod_checker(3, 1)
print(mod_3_1(4)) # True

 


Memory limit: 256 Mb
Time limit: 2s
