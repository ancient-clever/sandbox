Character by character match

The simplest example of a pattern is a string that contains the characters that we want to find. For example, the "abc" string is the pattern for the "abc" string, thus

import re
results = re.findall("abc", "abcAbcabc")
print(results)


Outputs all the occurrences of the "abc" string in the "abcAbcabc" string.

In this problem the input is text. Using regular expressions, find all occurrences of the "you" substring in it and display its number.

NB. This problem is case-sensitive.

NB. For the correct reading of the standard input in this problem you need to use the 


import sys
data = sys.stdin.read()


Since the text can have multiple lines, and input() reads on a by-line basis.

What are you doing here?
Sample Output:
1
