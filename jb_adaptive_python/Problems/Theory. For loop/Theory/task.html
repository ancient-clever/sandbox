For loops iterate over a given sequence. Here is an example: 
primes = [2, 3, 5, 7]
for prime in primes:
    print(prime)
 
For loops can iterate over a sequence of numbers using the 'range' function: 
>>> ﻿for x in range(3, 6):
...    print(x)
3
4
5
