Let use review the Loggable class:


  

  
import time
class Loggable:
    def log(self, msg):
        print(str(time.ctime()) + ": " + str(msg))

  

   

  

  
 
   
It has the exactly one method log, which allows outputting to the log (in this case in stdout) some message, adding the current time.
 
 
   
Implement the class LoggableList, inheriting it from the classes list and Loggable, so that when adding an element to the list using the append method, the message containing the just added element would be sent to the log.
 
   
Note
 Your program should not contain the Loggable class. Your program will have this class available during the check, and it will contain the log method, described above.
