Implement a data structure that represents an extended stack. It is necessary to support the pushing (appending) of the element to the top of the stack, the popping (removal) from the top of the stack, and also the operations of addition, subtraction, multiplication and integer division.

The operation of addition on the stack is defined as follows. The top element (top1) is removed from the stack, then the next top element (top2) is removed, and then the element equal to top1 + top2 is pushed to the stack.

In similar way are defined the operations of subtraction (top1 - top2), multiplication (top1 * top2) and integer division (top1 // top2).

Implement this data structure as the ExtendedStack class, by inheriting it from the standard list class.
