Continuous backpack

First line contain the number of items \( 1 \le n \le 10^3 \) and capacity of the backpack \( 0 \le W \le 2 \cdot 10^6 \). Each of the next \( n \) lines specifies the cost \( 0 \le c_i \le 2\cdot 10^6 \) and volume \( 0 \lt w_i \le 2\cdot 10^6 \) of an item (\( n \), \( W \), \( c_i \), \( w_i \) â€” integers). Output the maximum cost of parts of the items (you can separate any part from each item, its cost and volume will decrease proportionally), placed in the backpack, with an accuracy of not less than three decimal places.


Sample Input:
3 50
60 20
100 50
120 30

Sample Output:
180.000


Memory limit: 256 Mb
Time limit: 5s
