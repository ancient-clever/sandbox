A string is a sequence of characters. You can access the characters one at a time with the bracket operator: 
>>> fruit = 'banana'
>>> letter = fruit[1]

﻿The second statement selects character number 1 from \( fruit \) and assigns it to \( letter \). The expression in brackets is called an index. The index indicates which character in the sequence you want (hence the name). The index is an offset from the beginning of the string, and the offset of the first letter is zero. So: 
>>> print(letter)
﻿a
