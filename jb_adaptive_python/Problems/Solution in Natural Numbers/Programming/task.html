On input given the number \( N \) (\( 0 \leqslant N \leqslant 10^{12} \)). If there exists natural numbers (or 0) \( x \), \( y \) such that \( x^2 + y^2 = N \), then you need to output them. In case there are several solutions, output any of the pairs. Otherwise, output "No solution".


Sample Input:
29
Sample Output:
2 5

Sample Input:
7
Sample Output:
No solution


Memory limit: 256 Mb
Time limit: 1s
