Given two strings s and t, consisting of lowercase letters of the Latin alphabet.

Find the number of occurrences of the line t in the line s.

Example:

s = "abababa"
t = "aba"

There are 3 occurrences of the line t in the line s:

abababa
abababa
abababa

Sample Input:
abababa
aba
Sample Output:
3


Sample Input:
abababa
abc
Sample Output:
0


Sample Input:
abc
abc
Sample Output:
1


Sample Input:
aaaaa
a
Sample Output:
5



Memory limit: 256 Mb
Time limit: 2s
