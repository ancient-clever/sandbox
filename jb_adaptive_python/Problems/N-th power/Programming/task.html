It is possible to find a n-th power much quicker than by making n multiplications! To do this you need to use the following recurrence relations:

\( a^n = (a^2)^{n/2} \) for even n,

\( a^n = a × a^{n−1} \) for odd n.

Implement the algorithm of quick exponentiating using a recursion function.

Input data format

Given a real number a and a non-negative integer n.


Sample Input:
2.0
1
Sample Output:
2

Sample Input:
2
2
Sample Output:
4


Memory limit: 256 Mb
Time limit: 1s
