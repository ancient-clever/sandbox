Write the function update_dictionary(d, key, value), which takes the dictionary \( d \) and the two numbers \( key \) and \( value \).

If the key \( key \) is present in the dictionary \( d \), then append the value \( value \) to the list, which is stored by this key.
If the key \( key \) is not present in the dictionary, you need to append the value to the list keyed \( 2â‹…key \). Is there is no key \( 2\cdot key \), you need to add the key \( 2\cdot key \) to the dictionary and store the list \( [value] \) by it.

You need to implement only this function, there should be no code except of it.
The function should not call the input and print functions.

An example of how function works:

d = {}
print(update_dictionary(d, 1, -1))  # None
print(d)                            # {2: [-1]}
update_dictionary(d, 2, -2)
print(d)                            # {2: [-1, -2]}
update_dictionary(d, 1, -3)
print(d)                            # {2: [-1, -2, -3]}


Memory limit: 256 Mb
Time limit: 5s
