Repeats

In this problem, we will look on how to implement the symbol repetition.

We can use the three different structure to implement the repeat:

  1.  Wildcards { }, inside which you can locate the minimum/maximum number of repeats, which satisfy our needs, or the exact number of repeats that we look for. For example, the template "A{3, 5}" fits all substrings, consisting of consecutive symbols –ê, with length from 3 to 5, and the template "T{4}" fits only those substrings, which contain 4 T in a row.
  2. Wildcard *. This wildcard means any number of occurrences of the character (or a group of characters): from zero to any desired number. Thus, the template "A*" fits all sequences consisting of any amount of A in a row, and an empty string (as zero also fits).
  3.  Wildcard +. This wildcard means any positive number of occurrences of the character (or a group of characters): from one to unlimited. Thus, the template "A+" fits all sequences consisting of any amount of consecutive A. Unlike "A*" - an empty string will not fit.

All wildcards of the repeat are applied to the character, after which they recorded (or a group of characters).

Usage examples:

import re
results = re.findall("A+", "AAATATAGCGC")
print(results)
results = re.findall("TA{2}", "AATATAGCGTATA")
print(results)
