Write a program that outputs "solved" field, i.e., for each cell, which is not a mine, it indicates the number of mines in the neighbouring cells (including diagonal directions)

Input format:
The first line contains two natural numbers \( 1 \le n,m \le 100 \), after which the \( n \) lines contain a description of the field in the form of a sequence of dots '.' and asterisks '*', where the dot denotes an empty cell, and the asterisk â€“ the cell with a mine.

Output format:
\( n \) lines of the field, each cell of which will contain either a number from 0 to 8, or a mine (marked by "*"), wherein any number means the number of mines in the neighbouring cells.


Sample Input:
4 4
..*.
**..
..*.
....
Sample Output:
23*1
**32
23*1
0111


Memory limit: 256 Mb
Time limit: 5s
