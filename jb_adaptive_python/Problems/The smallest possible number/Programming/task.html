You should write a program which creates the smallest possible number from the digits of the two given natural numbers, while maintaining the order of the digits in these given numbers.

Input data

Input string contains the two natural numbers, separated by a space. The numbers are greater than zero and less than 10255.

Output data

You should output the smallest possible number, described in the problem statement.


Sample Input:
41238 345

Sample Output:
34123458


Memory limit: 256 Mb
Time limit: 5s
