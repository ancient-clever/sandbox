Calculator

You have a basic calculator, which has only three operations with the current number \( x \): replace \( x \) by \( 2x \), \( 3x \) or \( x+1 \). Given the integer \( 1 \le n \le 10^5 \), find the minimum number of \( k \) operations, necessary to get \( n \) from 1. Output \( k \) and the sequence of intermediate numbers.


Sample Input:
1

Sample Output:
0
1

Sample Input:
5

Sample Output:
3
1 2 4 5

Sample Input:
96234

Sample Output:
14
1 3 9 10 11 22 66 198 594 1782 5346 16038 16039 32078 96234


Memory limit: 256 Mb
Time limit: 5s
