Koch curve is a simple geometric fractal.

This fractal is constructed as follows: we take an interval and divide it into three equal parts. Instead of the middle part we insert two the same intervals, set at an angle 60 degrees to each other (see the image, transition from \( n=0 \) to \( n=1 \)).

This process is repeated at every iteration: each interval is replaced by the four intervals.


Write a program that takes number n as input − the number of iterations to generate curve, and outputs the sequence of turn angles when drawing a corresponding line from the start to the end point, without leaving the pen.

Input format:

Single line with the integer \( n \), \( 1\le n \le 10 \).

Output format:

Sequence of turn. Each turn is specified in a separate line as a word "turn" and the angle of rotation. The angle of rotation should be in degrees in the interval [-180; 180].

Addition:

Python includes the turtle library, allowing to draw with the help of so-called "turtle".

We will need several simple commands: 

turtle.forward(length)
turtle.left(degree)

Here is a simple script that draws the Koch curve for the specific n: 
import turtle
def koch_turns(n):
    ... # here should be the implementation of the function,            
        # generating the sequence of numbers, setting the line turns.  
def turtle_koch_curve(n, line_length=10):
    for move in koch_turns(n):
        turtle.forward(line_length)
        turtle.left(move)
    turtle.forward(line_length)


Call the function turtle_koch_curve with the argument n (number of iterations), and you will see how the turtle draws the fractal. You can also specify a length of the segment if necessary.

If an error occurs, try installing the tk module.

Please note than the function koch_turns should return a list of integers, for example:

print(list(koch_turns(1)))  # [60, -120, 60]



Sample Input:
1
Sample Output:
turn 60
turn -120
turn 60
