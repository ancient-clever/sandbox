In this problem, you have to write the fibonacci_sequence class, from which we can take the iterator, which will sort numbers from the Fibonacci sequence.

The Fibonacci sequence is defined like below. The first two of its elements are equal to 1, and each of the next elements is equal to the sum of two previous ones. The beginning of this sequence looks like the following:

1
1
2 (1 + 1)
3 (2 + 1)
5 (3 + 2)
8 (5 + 3)
13 (8 + 5)
...

The fibonacci_sequence class must have the constructor with one argument - the length of the sorted Fibonacci sequence.

Thus, iter(fibonacci_sequence(5)) should correspond to the iterator, which displays the first five numbers from the Fibonacci sequence each after each, and in case of attempts to obtain the next number after all the five numbers were displayed, an exception StopIteration should be thrown.



Memory limit: 256 Mb
Time limit: 2s
