Problem: read the data from the file and find their average (mean) values.

As input, your solution will get an address, where the csv-file is located â€“ you have to read data from that file. The first line in the file - column names, other lines - numeric data (that is, each line except the first one, contains a sequence of real numbers, separated by commas).

Find and print the vector of average values along the columns of the input data. Thus, if the input data file looks like below:

a,b,c,d
1.5,3,4,6
2.5,2,7.5,4
3.5,1,3.5,2


the solution will be

[ 2.5  2.   5.   4. ]


You can pass a file object as a file in np.loadtxt. This is convenient in the cases such as now, when the data is not on your computer, but somewhere in the network. How to download them? Using the standard libraries:

>>> from urllib.request import urlopen
>>> f = urlopen('https://stepic.org/media/attachments/lesson/16462/boston_houses.csv')

Now f contains the file object, which you can pass to np.loadtxt.
