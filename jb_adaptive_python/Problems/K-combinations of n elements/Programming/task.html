You need to generate all possible k-combinations of n elements.

Input data format:
Two numbers (k and n) separated by a space. It is guaranteed that \( 0 \leq k \leq n \).

Output data format:
All the necessary number of lines, each of which contains k different numbers between 0 and n-1 inclusively, separated by a space.


Sample Input:
2 3
Sample Output:
0 1
0 2
1 2

Sample Input:
1 1
Sample Output:
0


Memory limit: 256 Mb
Time limit: 5s
