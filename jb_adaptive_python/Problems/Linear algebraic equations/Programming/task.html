Write a program that solves system of linear algebraic equations by the Gaussian method.

Input data format:

The first line specifies two numbers: the number of equations \( n \) (\( n \ge 1 \)) and the number of indeterminates \( m \) (\( m \ge 1 \)). Next goes n number of lines, each of which contains the m+1 number. First m numbers are the coefficients of the i-th equation of the system, and the last, (m+1) number is the coefficient \( b_i \), placed in the right part of the i-th equation.

Output data format:

The first line should output the word YES, if the solution exists and it is the only one, the word NO if the solution does not exist, and the word INF in case if there are infinitely many solutions. If the solution exists and it is the only one, you should provide the solution of the simultaneous linear algebraic equations in the form of m numbers in the second line, separated by a space.


Sample Input:
3 3
4 2 1 1
7 8 9 1
9 1 3 2

Sample Output:
YES
0.2608695652173913 0.04347826086956526 -0.1304347826086957

Sample Input:
2 3
1 3 4 4
2 1 4 5

Sample Output:
INF

Sample Input:
3 3
1 3 2 7
2 6 4 8
1 4 3 1

Sample Output:
NO


Memory limit: 256 Mb
Time limit: 5s
