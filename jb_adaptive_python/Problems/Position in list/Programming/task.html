Write a program, which reads the list of numbers \( lst \) from the first line and the number \( x \) from the second line of the input; and outputs all positions, on which the number \( x \) can be found in the given list \( lst \).

The positions are numbered from zero; if the number \( x \) is not present in the list - output "Missing" (without quotation marks, starting from the capital letter).

Output the positions in a single line, by the ascending absolute values.


Sample Input:
5 8 2 7 8 8 2 4
8
Sample Output:
1 4 5

Sample Input:
5 8 2 7 8 8 2 4
10
Sample Output:
Missing


Memory limit: 256 Mb
Time limit: 5s
