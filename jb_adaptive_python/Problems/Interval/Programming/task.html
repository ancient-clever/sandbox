Interval

Given an integer as input. Output True if its value is within the interval \( (-15, 12] \cup (14, 17) \cup [19, +\infty) \), and False otherwise (case sensitive).

Please note the different brackets, which are used to specify intervals. The problem uses semi-open and open intervals. You can read more about it on the Wikipedia.

Sample Input:
20
Sample Output:
True

Sample Input:
-20
Sample Output:
False


Memory limit: 256 Mb
Time limit: 5s
