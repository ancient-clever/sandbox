The first line contains the integer \( N \) (\( 1 \leq N \leq 10^5 \)). Next specified the array of integers from \( -10^9 \) to \( 10^9 \). Further goes the number of requests \( K \) (\( 1 \leq K \leq 10^5 \)), next go \( K \) pairs of requests \( L \), \( R \)  (\( -10^9 \leq L \leq R \leq 10^9) \). You need to output the \( K \) numbers, which are the responses to the requests "How many numbers in the array are located on the range from \( L \) to \( R \)?".


Sample Input:
5
1 4 2 22 12
3
2 5
3 8
30 33

Sample Output:
2 1 0


Memory limit: 256 Mb
Time limit: 5s
