The sequence of \( n \gt 0 \)  integers is called a jolly jumper if the absolute differences of the consecutive elements take all the possible values between \( 1 \) and \( n-1 \).

For example, the sequence 1 -3 -4 -1 1 is a jolly jumper sequence, as the absolute differences are equal to 4 1 3 2, accordingly, and \( n-1 = 4 \).
We assume that a sequence containing only one element is a jolly jumper.

Write a program that checks whether the entered sequence is a jolly jumper.

Input format:
A string, containing \( 1 \le n \le 10000 \) space separated integers.

Output format:
One line, displaying "Jolly" (without quotation marks), if the sequence is a jolly jumper and "Not jolly" if otherwise.


Sample Input:
1 -3 -4 -1 1
Sample Output:
Jolly

Sample Input:
1 3 5
Sample Output:
Not jolly

Sample Input:
4
Sample Output:
Jolly


Memory limit: 256 Mb
Time limit: 5s
