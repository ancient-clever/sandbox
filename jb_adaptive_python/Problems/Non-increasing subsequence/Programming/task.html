Non-increasing subsequence

Given \( n \) integers \( a_i \) from \( 1 \) to \( n \). Your task \( - \) to find the longest non-increasing subsequence of this sequence, i.e. such sequence of indices  (\( i_1 \lt i_2 \lt \dots \lt i_k \)), such that \( a_{i_1} \ge a_{i_2} \ge \dots \ge a_{i_k} \) and there is no sequence with the same properties of length \( k + 1 \).

Input format

In the first line contains the integer \( n \) \( - \) the number of elements in the sequence (\( 1\le n \le 100\,000 \)). The next line contains \( n \) integers from \( 1 \) to \( n \) \( - \) \( a_i \).

Output format

On the first line output the length of the sequence \( k \). On the next line output \( i_1, i_2, \dots, i_k \) (\( 1 \le i_1 \lt i_2 \lt \dots \lt i_k \le n \)). If there are several solutions with the optimal length, output any of them.

Hints for the solution

Use dynamic programming. This problem should be solved for \( O(n \log n) \).


Sample Input:
5
5 3 4 4 2

Sample Output:
4
1 3 4 5


Memory limit: 256 Mb
Time limit: 5s
