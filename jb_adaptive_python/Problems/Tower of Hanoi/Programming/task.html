Tower of Hanoi is one of the well-known puzzles with the following rules.

Given three rods (let's number them as 1, 2 and 3). The first rod has \( n \) rings on it with radius from 1 to \( n \). Rings are ordered by radius, and the largest of the rings lies below all others.

You should write an algorithm to move all \( n \) rings from the first rod to the third rod, sticking to the following rules:

  1. Only one disk can be moved at a time.
  2. No disk may be placed on top of a smaller disk.

The program should output the shortest sequence of actions required to move all the rings from the first rod to the third one.

Input format:
A positive integer \( n \).

Output format:
The list of moves to solve the puzzle. Each move should be written in a separate line in the format  "number of the rod from which we remove the ring" - "number of the rod, on which we put the ring"
(see example)


Sample Input:
2
Sample Output:
1 - 2
1 - 3
2 - 3

Sample Input:
1
Sample Output:
1 - 3


Memory limit: 256 Mb
Time limit: 5s
