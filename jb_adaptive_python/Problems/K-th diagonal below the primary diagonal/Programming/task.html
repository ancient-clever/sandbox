Given the square two-dimensional matrix of size n × n and the number k. Output the elements of the k-th diagonal below the primary diagonal (i.e. if k = 1, you need to output the elements of the first diagonal below the primary one, if k = 2 - the second diagonal, etc.).

The value of k can be negative, for example, if k = −1, you need to output the value of the first diagonal above the primary one. If k = 0, it is needed to output the elements of the primary diagonal.

The program gets the number n as input, not exceeding 10, than the matrix of size n × n, than the number k.


Sample Input:
4
1 2 3 4
5 6 7 8
0 1 2 3
4 5 6 7
1
Sample Output:
5 1 6

Sample Input:
4
1 2 3 4
5 6 7 8
0 1 2 3
4 5 6 7
-2
Sample Output:
3 8


Memory limit: 256 Mb
Time limit: 1s
