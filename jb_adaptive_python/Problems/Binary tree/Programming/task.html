Given the number \( N \lt 10^6 \) and the sequence of integers from the interval \( [-2^{31}..2^{31}] \) of length N.

Construct the binary tree, specified by the naive insertion order.

Meaning: when adding the next number K to the tree with the root “root”, if the root→Key ≤ K, then the K node is added to the right root subtree; otherwise – to the left root subtree.

Output elements in a post-order (bottom-up), separating them by spaces.


Sample Input:
3
2 1 3
Sample Output:
1 3 2


Memory limit: 256 Mb
Time limit: 5s
