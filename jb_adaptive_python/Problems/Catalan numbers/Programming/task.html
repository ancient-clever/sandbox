Knowing the explicit form of the Catalan numbers, output the recurrence relation, expressing \( C_{n+1} \) only through \( C_n \) and use it for the effective calculation of \( C_n \). Input gets the number \( n \leq 100000 \). Output \( C_n \) by modulo \( 1000000007 \).

You may need the operation "modulo division ". Use the fact that for a simple \( p \) integer division can be replaced by multiplication by the following formula:

\( \frac{a}{b} \equiv a * b^{p-2} (\mod p). \) 


Sample Input:
99009
Sample Output:
485259624


Memory limit: 256 Mb
Time limit: 5s
