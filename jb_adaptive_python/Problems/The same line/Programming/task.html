Write a program, which reads the line from a standard input, using the input() function, and outputs the same line to the standard output, using the print() function.

Please pay attention that you need to use the input function without any parameters, and that you need to output only that, what was transferred to the input of this program.

Input format:
A line containing arbitrary characters, including spaces.

Output format:
Single line, containing the line from the input.


Sample Input:
Hello World!
Sample Output:
Hello World!

Sample Input:
Word
Sample Output:
Word


Memory limit: 256 Mb
Time limit: 5s
