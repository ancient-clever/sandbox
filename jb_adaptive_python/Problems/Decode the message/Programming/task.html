We have the following symbol mappings:

  • A ↔ 00
  • C ↔ 01
  • G ↔ 10
  • T ↔ 11

Given a string of 0's and 1's, decode the message via the above mapping and print out the resulting decoded message to standard output.


Sample Input:
00011011
Sample Output:
ACGT


Memory limit: 256 Mb
Time limit: 5s
