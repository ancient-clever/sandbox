Grouping

In this problem, we will deal with the implementation of a group of symbols.

Wildcards "( )" are used to group symbols.

Groups of symbols allow us using repeats on sequences: this way, the "(ab)+" template will fit for any string like "ababababab...ab".

In addition, symbols grouping allows us to find an interesting substrings inside a long pattern: for example, if we know that somewhere inside the string is an "ID: 254" substring, then by using the "ID: (\d+)" pattern (\d means any number) we will be able to get only the number 254 we are interested in:

import re
string = "ID: 254, ID: 14, ID: 88"
ids = re.findall("ID: (\d+)", string)
print(ids)

It is important to understand that the findall function from the re module returns a tuple from all groups, which it found; and if the groups are not obviously defined, it considers the whole pattern as one group. If we change the pattern to "(ID: (\d+))", the function returns tuples with the two elements: the whole pattern and its numeric ID.

import re
string = "ID: 254, ID: 14, ID: 88"
ids = re.findall("(ID: (\d+))", string)
print(ids)


NB. It is important to understand that the groups will be listed within the tuple in the order of their occurrence in the pattern. (In the order of occurrence of the respective groups’ opening parentheses)
