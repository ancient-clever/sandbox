You are given a sequence of integers, and you need to process it and output the sum of the first five numbers of the sequence, then the sum of the second five numbers, and so on.

However, consistency is not given to you all at once. With the course of time, you get its subsequent parts. For example, the first three elements, then the next six elements, then the next two ones, etc.

Implement the Buffer class, which will store the sequence elements and output the sum of the five consequent elements as accumulation proceeds.

One of the requirements for the class is that it should not store more elements than it is necessary, that is it does not have to store the elements that are already included in the five, for which the sum was displayed.
