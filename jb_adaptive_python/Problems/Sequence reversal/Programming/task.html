Write a program, which turns a sequence of integers \( x_1, x_2, \ldots, x_n \) into the sequence \( (x_1 + x_n), (x_2+x_{n-1}), (x_3 + x_{n-2}), \ldots \) of length \( \lceil n/2 \rceil \).

Input format: input is a positive integer \( n\lt 10^6 \), next go the \( n \) integers separated by spaces, corresponding to \( x_1,\ldots,x_n \).

Output format: the output should be the \( \lceil n / 2 \rceil \) space-separated integers, corresponding to the sequence \( (x_1 + x_n), (x_2+x_{n-1}), (x_3 + x_{n-2}), \ldots \). In the case if number \( n \) is an odd one, \( x_{(n + 1)/2} \) (i.e. the middle number in the array) should serve as the last element of the sequence.


Sample Input:
10 30 32 43 65 -32 54 34 -23 11 9
Sample Output:
39 43 20 99 22


Memory limit: 256 Mb
Time limit: 5s
