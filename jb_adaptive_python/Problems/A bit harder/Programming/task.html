A bit harder

Let there be an allowed multiple assignments in a form of:

string = "14 15 16"
a, b, c = string.split(" ")

It would be useful to highlight all variables within this assignment!

Also a code can have some comments (!), inside which the assignment doen't happen, and it would be great to ignore assignments inside these comments.

Let me remind that a comment starts from the # symbol and continues to the end of the line. For example:

a, b, c = string.split(" ") # multiple assignment
# a, b, c = string.split(" ") multiple assignment
We are interested only in the first line.

Assignments may also be for the attributes. That is why a variable may contain dots, but may not start from the dot and may not contain two dots in a row.

The input of this problem in a Python code; each line may contain one assignment, but with a several variables. You need to find all uncommented assignments and for each of them output the space separated number of the line (in which it occurs) and the name of each variable (in the same order as they are in the assignment).

NB. This problem may require using re.match instead of re.findall; the re.match function checks whether the string matches the pattern (does not find the occurrence of the pattern in the string, like re.findall does), the re.match function returns MatchObject if the string (or the beginning of the string) matches the pattern, and None otherwise. MatchObject has the groups() method, which contains the grouped symbols. An example of the use of re.match:

import re
pattern = "(ID: (\d+))"
str1 = "ID: 5853"
str2 = "xID: 455"
match1 = re.match(pattern, str1)
match2 = re.match(pattern, str2)
print(match1)
print(match1.groups())
print(match2)

Of course, a link to documentation https://docs.python.org/3.4/library/re.html#re.regex.match
